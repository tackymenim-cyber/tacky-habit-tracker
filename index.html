🚀 كيف تحصل على بيانات Firebase؟
تحتاج إلى إنشاء مشروع مجاني على Firebase Console. اتبع الخطوات:
  1. اذهب إلى console.firebase.google.com وأنشئ مشروعاً جديداً
  2. من القائمة الجانبية → Build → Authentication → Get started → فعّل Google كطريقة تسجيل دخول
  3. من القائمة الجانبية → Build → Firestore Database → Create database → اختر Start in test mode
  4. اضغط على أيقونة الترس ⚙️ → Project settings → مرّر لأسفل → اضغط </> (Web app) → سجّل التطبيق
  5. انسخ كامل الكود firebaseConfig = { ... } والصقه في الخطوة التالية
